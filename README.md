# 🎮 Simon Says Game

A fun and interactive **memory-based color sequence game** built using **HTML**, **CSS**, and **JavaScript**.

Test your memory and reflexes as you follow the flashing color patterns —  
each level gets trickier and faster! 🔥

---

## 🧩 **How to Play**
1. Click the **Start** button to begin the game.
2. Watch carefully — a random color button will flash.
3. Repeat the exact sequence by clicking the buttons in the same order.
4. Each successful level adds a new color to the sequence.
5. One mistake, and the game resets — how far can you go?

---

## 🧠 **Features**
- 🎯 Randomized pattern generation  
- 🌈 Smooth animations and transitions  
- 🧮 Level-based difficulty increase  
- 💥 Flashing effects for both system and user input  
- 💫 Restart anytime with a single click  

---

## 💻 **Tech Stack**
- **HTML5** for structure  
- **CSS3** for styling and animations  
- **Vanilla JavaScript (ES6)** for logic and interactivity
